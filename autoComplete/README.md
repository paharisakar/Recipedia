ajax-search-box
===============

Ajax Search box using Node and MySQL

Download the code and type
<code>npm install</code>

Run the code using 
<code>node Server.js</code>

Live demo : https://www.youtube.com/watch?v=6xEmZGHIHP4&feature=youtu.be
